# 프로젝트 문서 관리툴 — 업데이트 노트

---

## V0.84 (2026-04-11)

### 버그 수정 및 안정화

#### 🔧 주요 버그 수정
- **문서 내용 교체** 탭의 **찾기(🔍)** 기능이 제대로 작동하지 않던 문제 해결  
  → 실제 파일 전체 경로(`UserRole`)를 사용하도록 수정하여 검색 정확도 대폭 향상
- **파일 이름 교체**(`run_rename`)가 정상 작동하지 않던 문제 수정
- **되돌리기(↩)** 기능 강화  
  - 파일 이름 되돌릴 때 리스트 표시와 실제 파일 경로가 동시에 정상 복구되도록 개선  
  - 반복 되돌리기 후 파일 재추가 시 발생하던 중복/추가 실패 문제 해결
- `FileListWidget` 내부 경로 관리 로직 보강 (안정성 향상)

#### 📋 기타 개선
- `run_rename`, `undo_action`, `run_find` 등 핵심 함수 로직 정리 및 안정화
- 불필요한 코드 중복 제거 및 예외 처리 강화
- 사용자 피드백 기반으로 가장 자주 발생하던 버그들을 집중적으로 잡음

### 기존 기능
- V0.83에서 추가된 **파일명만 표시 + 툴팁 전체 경로**, **우클릭 컨텍스트 메뉴** 등 모든 기능 정상 유지

---

**전체적으로 프로그램 안정성이 크게 향상되었습니다.**  
특히 문서 내용 찾기와 파일 이름 교체, 되돌리기 기능이 이제 제대로 동작합니다!

## V0.83 (2026-04-10)

### 신규 기능 및 개선사항

#### 📂 파일 목록 표시 방식 최적화
- 파일 추가 시 전체 경로 대신 **파일명만 표시**되도록 개선하여 가시성 확보
- 추가한 파일리스트 개수 표시
- 실제 파일 경로는 내부 데이터(`UserRole`)에 저장하여 기능적 동작은 유지
- 목록 아이템에 마우스를 올리면(Hover) 전체 경로를 확인할 수 있는 **툴팁(Tooltip) 기능** 추가

#### 🖱 우클릭 컨텍스트 메뉴 강화
- 파일 리스트 항목 우클릭 시 사용할 수 있는 편의 기능 추가:
    - **경로 복사**: 선택한 파일의 전체 경로를 클립보드에 복사
    - **목록에서 제거**: 선택한 항목을 리스트에서 삭제 (실제 파일은 삭제되지 않음)
    - **폴더 열기**: 해당 파일이 위치한 상위 폴더를 탐색기로 즉시 열기


## V0.82 (2026-04-10)

### 신규 기능 및 개선사항

#### 📂 비동기 폴더 스캔 (Background Worker)
- 대량의 파일을 포함한 폴더 추가 시 UI가 멈추던 "응답 없음" 현상 해결
- `FolderScanWorker` 스레드를 도입하여 백그라운드에서 실시간 파일 검색 및 목록 업데이트 수행
- 파일 목록 로드 중 하단 상태바를 통해 진행 상황 표시

#### 📏 파일 목록 UI 최적화
- 파일 리스트 항목(`FileListWidget`)의 행간 간격을 조밀하게 조정 (높이 20px 고정)
- 불필요한 내부 여백(Padding) 및 외부 간격(Spacing) 제거로 가시성 향상
- 텍스트가 잘리지 않으면서도 한 화면에 더 많은 파일 경로를 볼 수 있도록 개선

#### 🔢 파일 개수 표시 개선
- '문서 내용 교체' 탭과 '파일명 수정' 탭 모두에 실시간 파일 카운트 라벨 추가
- 파일 추가/삭제 시 대상 파일 개수가 즉시 업데이트되도록 개선

### 버그 수정 및 코드 정리
- `FileListWidget`에서 중복 파일 체크 시 발생하던 `all_paths` 참조 오류(AttributeError) 수정
- 중복 선언된 `import` 구문 정리 및 스레드 클래스 구조 최적화
- `FolderScanWorker`가 가비지 컬렉터에 의해 도중 종료되지 않도록 참조 방식 수정

## V0.80  (2026-04-10)

### 신규 기능

#### 검색 결과 창 개선
- 기존의 단순 텍스트 팝업 대신, 파일 목록을 선택할 수 있는 리스트 형태로 변경
- 검색된 파일명 및 발견 개수를 항목별로 표시
- 항목 선택 후 아래 버튼으로 바로 조작 가능

#### 📂 열기 버튼
- 선택한 파일을 편집기로 바로 열기
- 기본값: 메모장(Windows) / 시스템 기본 앱
- 설정에서 편집기를 등록한 경우 해당 편집기로 실행
- 항목 더블클릭으로도 동일하게 동작

#### 📋 파일명 복사 버튼
- 선택한 파일의 전체 경로를 클립보드에 복사
- 복사 완료 시 하단 상태바에 알림 표시

#### ✏ 편집기 등록 (설정 메뉴)
- 메뉴 → ⚙ 설정 → ✏ 편집기 등록
- 찾아보기로 원하는 편집기 실행파일(.exe) 직접 선택
- 초기화 버튼으로 메모장 기본값으로 복귀
- 설정값은 config.json에 자동 저장

### 버그 수정
- `get_button_style()` 내 실행되지 않는 죽은 코드 제거

## V0.72 이전 — 기존 기능 목록

### 📂 문서 내용 교체
- 선택한 파일 안의 특정 텍스트를 찾아 일괄 교체
- 텍스트 기반 파일(md, txt, csv, html 등)만 수정 가능
- hwp, doc, docx, xlsx 등 바이너리 파일은 내용 수정 불가

### 📝 파일 이름 교체
- 기존 문자 → 변경 문자로 파일명 일괄 교체

### 🔢 버전 변환
- 파일명의 언더바를 버전 점(.)으로 변환 (예: v1_2 → v1.2)

### 🔤 공백 변환
- 파일명의 언더바를 공백으로 변환

### ✂ 묶인곳 지우기
- 시작~끝 문자 사이의 텍스트 삭제 (예: (임시) → 삭제)
- 전부 삭제 / 첫 번째만 삭제 선택 가능

### 🔢 숫자 남기기/삭제
- 파일명에서 숫자만 남기거나 모두 제거

### 🔣 자리수 맞추기
- 숫자 부분에 0을 붙여 자리수 통일 (예: 3 → 003)
- 앞/뒤 맞춤, 초과 시 처리 방식 선택 가능

### 🔖 번호 붙이기
- 파일 이름 앞/뒤 또는 폴더별로 일련번호 부여
- 자리수, 시작값, 구분자, 위치 설정 가능
- 기존 번호 교체 모드 지원

### ↩ 되돌리기
- 가장 최근 교체 작업을 이전 상태로 복구
- 내용 교체 / 이름 교체 탭별로 독립 동작

### ⚙ 중복 이름 처리
- 덮어쓰기 / 뒤에 숫자 붙이기 / 건너뛰기 중 선택

### 🎨 테마 변경
- 색상 항목별 커스텀 설정
- 다크 블루 / 시스템 테마 / 기본값 초기화 지원

### ⌨ 단축키
| 단축키 | 기능 |
|---|---|
| Ctrl + Z | 되돌리기 (현재 탭 기준) |
| Ctrl + F | 찾기 (문서 내용 탭) |
| Ctrl + R | 교체 실행 (현재 탭 기준) |
| Ctrl + D | 목록 초기화 (현재 탭 기준) |
| Ctrl + A | 목록 전체 선택 |
| Delete | 선택 항목 삭제 |

### 기타
- 파일/폴더 드래그앤드롭 지원
- 우클릭 컨텍스트 메뉴 (삭제, 전체선택, 초기화)
- 설정 config.json 자동 저장/로드
