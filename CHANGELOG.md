## V0.9 (2026-04-19)

### 파일 이름 탭 전면 재설계

#### 🏗 구조 개편
- 이중 구조(`FileListWidget` + `QTreeWidget`) 제거 → `RenameTreeWidget` 단일 소스로 통합
- 트리 뷰 [파일 경로 / 파일 이름] 2컬럼으로 단순화
- 동기화 버그 근본 해결

- [적용하기] 버튼 제거, 모든 버튼 즉시 실행
- [되돌리기(전단계)] / [되돌리기(원래이름)] 유지

- 파일 이름 지우기 (001 형식 3자리 자동 부여)
- 위치 지우기 (구간 삭제, 앞/뒤 방향 선택)
- 이름 앞에 추가 / 이름 뒤에 추가 (직접입력·폴더명·일시 선택)
- 확장자 변경/삭제
- 번호 붙이기 (버튼으로 추가)

#### 🎨 탭 UI
- 문서내용 탭 파란계열 고정 / 파일이름 탭 주황계열 고정 (테마 무관)
- 활성 탭 크게 / 비활성 탭 작게

#### 🔧 버그 수정
- 목록 초기화 정상 작동
- save_config 예외처리 보강
- 숫자 삭제 시 빈 이름 → AAAB~ZZZZ 26진수 영문 방식 자동 부여
