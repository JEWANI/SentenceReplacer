# 프로젝트 문서 관리툴 — 업데이트 노트

---

## V0.9 (2026-04-19)

### 파일 이름 탭 전면 재설계

#### 🏗 구조 개편
- 기존 이중 구조(`FileListWidget` + `QTreeWidget`) 완전 제거
- `RenameTreeWidget` 단일 데이터 소스로 통합 → 동기화 버그 근본 해결
- 트리 뷰 **[파일 경로 / 파일 이름]** 2컬럼 표시로 단순화
- 드래그앤드롭 트리에서 직접 처리

#### ⚡ 즉시 실행 방식으로 전환
- 기존 **[적용하기]** 버튼 제거
- 모든 버튼 클릭 시 실제 파일 즉시 변경
- **[되돌리기(전단계)]** / **[되돌리기(원래이름)]** 유지

#### 🆕 신규 버튼 추가 (파일이름 탭)
- **파일 이름 지우기** — 경고 후 즉시 `001.확장자` 형식으로 일괄 변경 (3자리 0패딩)
- **위치 지우기** — 지정 위치 구간 문자 삭제 (앞/뒤 방향 선택)
- **이름 앞에 추가** — 직접입력 / 폴더명 / 변경일시 / 생성일시 선택
- **이름 뒤에 추가** — 동일
- **확장자 변경/삭제** — 확장자 변경 또는 빈칸으로 삭제
- **번호 붙이기** — 기존 메뉴 기능을 버튼으로 추가

#### 🎨 탭 UI 개선
- 문서내용 탭: **파란계열 고정** (테마 무관)
- 파일이름 탭: **주황계열 고정** (테마 무관)
- 활성 탭: 글자 크게 + 패딩 크게 / 비활성 탭: 글자 작게 + 패딩 작게

#### 🔧 버그 수정 및 안정화
- 목록 초기화 버튼 정상 작동
- `save_config` 예외처리 보강 (실패 시 로그 출력)
- `add_path` 불필요한 `hasattr` 중복 체크 제거
- 숫자 삭제 시 결과가 빈 이름인 경우 → **AAAB~ZZZZ 26진수 영문 방식** 자동 부여
  - A~Z = 1~26, 기본 4자리, 초과 시 자동 확장 (예: `AAAB`, `AAAC`, ... `AAAZ`, `AABA` ...)

## V0.84 (2026-04-11)

### 버그 수정 및 안정화

#### 🔧 주요 버그 수정
- **문서 내용 교체** 탭의 **찾기(🔍)** 기능이 제대로 작동하지 않던 문제 해결  
  → 실제 파일 전체 경로(`UserRole`)를 사용하도록 수정하여 검색 정확도 대폭 향상
- **파일 이름 교체**(`run_rename`)가 정상 작동하지 않던 문제 수정
- **되돌리기(↩)** 기능 강화  
  - 파일 이름 되돌릴 때 리스트 표시와 실제 파일 경로가 동시에 정상 복구되도록 개선  
  - 반복 되돌리기 후 파일 재추가 시 발생하던 중복/추가 실패 문제 해결
- `FileListWidget` 내부 경로 관리 로직 보강 (안정성 향상)

#### 📋 기타 개선
- `run_rename`, `undo_action`, `run_find` 등 핵심 함수 로직 정리 및 안정화
- 불필요한 코드 중복 제거 및 예외 처리 강화

## V0.83 (2026-04-10)

### 신규 기능 및 개선사항

#### 📂 파일 목록 표시 방식 최적화
- 파일 추가 시 전체 경로 대신 **파일명만 표시**되도록 개선하여 가시성 확보
- 추가한 파일리스트 개수 표시
- 실제 파일 경로는 내부 데이터(`UserRole`)에 저장하여 기능적 동작은 유지
- 목록 아이템에 마우스를 올리면 전체 경로를 확인할 수 있는 **툴팁 기능** 추가

#### 🖱 우클릭 컨텍스트 메뉴 강화
- 파일 리스트 항목 우클릭 시 사용할 수 있는 편의 기능 추가:
  - **경로 복사**: 선택한 파일의 전체 경로를 클립보드에 복사
  - **목록에서 제거**: 선택한 항목을 리스트에서 삭제 (실제 파일은 삭제되지 않음)
  - **폴더 열기**: 해당 파일이 위치한 상위 폴더를 탐색기로 즉시 열기
